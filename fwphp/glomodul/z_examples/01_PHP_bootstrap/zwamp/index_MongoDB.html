14.8.2014 Install MongoDB 2.6.4 on win 8.1.1 64 bit,  Ver. 2.2+ does not support Windows XP.

http://www.mongodb.org

Download portable zip http://www.mongodb.org/dr//fastdl.mongodb.org/win32/mongodb-win32-x86_64-2008plus-2.6.4.zip/download
Upute za instalaciju http://docs.mongodb.org/manual/tutorial/install-mongodb-on-windows/
Do not make mongod.exe DB daemon visible on public networks without running in "Secure Mode" with auth setting. MongoDB is designed to be run in trusted environments, and DB does not enable “Secure Mode” by default."

  1. You can run portable MongoDB from any folder
    rename j:\mongodb-win32-x86_64-2008plus-2.6.4 J:\mdb
    *** cd J:\mdb ***
  2. md data\db moja mapa za podatke nije ispd root mape diska
  3. J:\mdb>bin\mongo.exe --help
    MongoDB shell version: 2.6.4 usage:
    bin\mongo.exe [options] [db address] [file names (ending in .js)]
    db address can be:
    foo foo DB on local machine
    192.169.0.5/foo foo DB on 192.168.0.5 machine
    192.169.0.5:9999/foo foo DB on 192.168.0.5 machine on port 9999

  4. J:\mdb>bin\mongod.exe ako je default data dir. (ispod J:), a meni nije, pa:
    bin\mongod.exe --dbpath j:\mdb\data
    MongoDB starting :
    pid=6552 port=27017 dbpath=\data 64-bit host=sspc
    journal dir=j:\mdb\data\journal
    allocating new datafile j:\mdb\data\local.ns, filling with zeroes...

  5. md "log"
    echo logpath="J:\mdb\log\mongo.log" > "mongod.cfg"

  6. Install MongoDB service
    "bin\mongod.exe" --config "mongod.cfg" --install'
    To use an alternate dbpath, specify path in config. file
    (e.g. J:\mdb\mongod.cfg) or on command line with --dbpath option.
    If dbpath directory does not exist, mongod.exe will not start.

    net start MongoDB
    net stop MongoDB
    "J:\mdb\bin\mongod.exe" --remove
    bin\sc.exe delete MongoDB
