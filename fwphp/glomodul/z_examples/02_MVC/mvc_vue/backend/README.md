# SimpleMVC

## 

**SimpleMVC** is a simple PHP MVC framework

Requirements ：

* PHP 5.6.0+

## Directories

```
project                 Root
├─app                   Applications
│  ├─controllers        Controllers
│  ├─models             Models
│  ├─views              Views
├─config                Configurations
├─core                  Framework Core files
├─static                Resources files : CSS, JS, Images
├─index.php             
```
